## IASA PORTAL
TS + Node.js + React + MongoDB로 만든 IASA PORTAL입니다.
## 사용법
1. secret 폴더를 복사합니다.
2. ```npm run build```로 파일을 빌드합니다.
3. ```npm run start```로 express 서버를 시작합니다.
